# Coding Rules

## Function Naming Convention

All Python function names MUST start with the prefix `fnE_`.

Examples of correct names:
- `fnE_calculate_total`
- `fnE_get_user`
- `fnE_process_data`

Examples of incorrect names (must be flagged in code review):
- `calculate_total` (missing `fnE_` prefix)
- `get_user` (missing `fnE_` prefix)
- `process_data` (missing `fnE_` prefix)

This rule applies to all functions defined with `def` in Python files.
