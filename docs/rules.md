# Coding Rules

## Function Naming Convention

All Python function names MUST start with the prefix `fnB_`.

Examples of correct names:
- `fnB_calculate_total`
- `fnB_get_user`
- `fnB_process_data`

Examples of incorrect names (must be flagged in code review):
- `calculate_total` (missing `fnB_` prefix)
- `get_user` (missing `fnB_` prefix)
- `process_data` (missing `fnB_` prefix)

This rule applies to all functions defined with `def` in Python files.
